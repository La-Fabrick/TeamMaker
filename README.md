# TeamManager

Create Teams and Manage Teams. Can be used as Library.

## Installation

Install the lib with gradle:
```gradle
repositories {
    maven { url 'https://jitpack.io' }
}
```
```gradle
dependencies {
    implementation 'com.github.La-Fabrick:TeamMaker:VERSION'
}
```

## Technologies

- Java 17
- Gradle 7.4
- ShadowJar
